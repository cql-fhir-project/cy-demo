## Cy-demo
Demonstrates abilities to translate, test and execute.

##### Disclaimer
This is not a production ready solution!

#### Requirements
1. install [git](https://git-scm.com/)
1. install [WebStorm](https://www.jetbrains.com/webstorm/download/)
1. install [NodeJS](https://nodejs.org/en/download/)
1. install [Java](https://www.java.com/en/download/)

#### Quick start
1. `git clone https://github.com/cql-fhir-project/cy-demo`
1. Open `cy-demo` in WebStorm
1. Open WebStorm terminal
1. save `.env-example` as `.env` file
1. Edit `.env`
1. `$ npm run i`
1. `$ npm run build`
1. `$ npm run execute`
1. `$ npm run start`
